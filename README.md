# 🚀 CryptoLens

A professional cryptocurrency dashboard built with HTML, CSS, and JavaScript.

CryptoLens provides real-time market data, trending cryptocurrencies, global market statistics, and a modern glassmorphism user interface powered by the CoinGecko API.

## ✨ Features

* Live cryptocurrency prices
* Top 10 cryptocurrencies by market cap
* Trending coins section
* Global market statistics
* Coin search functionality
* Responsive design
* Modern glassmorphism UI
* Auto refresh every 60 seconds
* Mobile-friendly layout
* Error handling and loading states

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* CoinGecko API
* Google Fonts

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Faizan-khan144/cryptolens-dashboard.git
```

2. Open the project folder

```bash
cd cryptolens-dashboard
```

3. Launch `index.html` in your browser
